Project Arduino

Kumpulan project Arduino paling sederhana. Fokus pada rangkaian LED, buzzer, dan kode singkat untuk mengenalkan pemula pada dunia Arduino. Sangat cocok sebagai langkah awal belajar mikrokontroler.

Lampu Polisi

Proyek sederhana menyalakan LED merah dan biru bergantian, meniru efek lampu rotator polisi. Cocok untuk pemula belajar Arduino.

Servo Bolak-Balek

Servo SG90 bergerak bolak-balik 0°–180° secara terus-menerus.

Lampu RGB

Arduino Uno dan LED RGB untuk menampilkan efek cahaya bergantian.
